# imageprocessor

A command line tool to ceasily crop and resize images

## Getting Started
* Install ImageMagick

Install the module with: `npm install imageprocessor`

```javascript
var imageprocessor = require('imageprocessor');
imageprocessor.awesome(); // "awesome"
```
